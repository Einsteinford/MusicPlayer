## MusicPlayer

#参照谷歌samples中的音乐播放器样式，制作的一个简单播放器demo

1.可实现上一首、下一首、暂停、播放、ListView选歌。

2.可拖动SeekBar对播放中的歌曲进行进度的调整。

3.拥有置于前台的服务，实现Notification的播放、暂停按钮与其他界面同步。

4.拥有桌面小部件，实现Widget的播放、暂停按钮与其他界面同步。
